How much time do I have to complete the course?

There is no finish date! You have life-time access to the course material so take as much time as you need to complete the course.

Do I need previous coding experience?

No previous coding experience is required to create this financial tracker. This beginner-friendly course guides you through each line of Python code and why it is used.

Is Python free to use?

Yes! Python is a free open source programming language and this course will provide a step-by-step walkthrough on how to get it installed onto your computer.

How hard is it to learn Python?

Python is considered one of the easiest programming languages for a beginner to learn. With the step-by-step code explanations in this course, building your financial tracker using Python will be easier than you may think!

What type of visualisations will I create?

In this course, you will learn how to visualise your expenses categories as both horizontal and vertical bar charts as well as pie charts and stacked bar charts.

How long is the course?

The course has over two hours of concise content. Each lecture contains detailed yet easy to follow walkthroughs that show you how to build your own financial tracker from scratch.
